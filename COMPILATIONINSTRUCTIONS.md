# Руководство по сборке проекта
Ниже приведена пошаговая инструкция по сборке проекта с помощью компилятора GCC.

## Сборка программы
Выполните следующие шаги для компиляции исходного кода:

**1. Откройте файл** `main.c` в любом текстовом редакторе для просмотра или редактирования исходного кода.

**2. Откройте терминал** в папке с файлом `main.c`.

**3. Введите команду компиляции:**
```
gcc main.c -o huffman.exe -Wall -Wextra
```
**4. Нажмите Enter** для выполнения команды.

**5. Готово!** Программа `huffman.exe` собрана и готова к запуску из консоли.

## Запуск программы
После успешной сборки выполните одно из следующих действий:

**_Запуск через терминал:_**
Введите в терминале команду:

```
./huffman.exe
```

**_Запуск через файловый менеджер:_**
Найдите исполняемый файл`huffman.exe` в корневой папке проекта и запустите его двойным щелчком мыши.

## Примечания
Флаги `-Wall` и `-Wextra` включают дополнительные предупреждения компилятора для улучшения качества кода.

При запуске через файловый менеджер в некоторых операционных системах может потребоваться подтверждение запуска исполняемого файла.
