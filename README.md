ИНСТРУКЦИЯ ПО ТЕСТИРОВАНИЮ:

1. КОМПИЛЯЦИЯ:
    - В CLion: Build > Build Project (или Ctrl+F9)
    - Вручную: gcc main.c -o huffman.exe

2. ЗАПУСК:
   Способ 1: Запуск с меню (рекомендуется)
    - Запустите программу без аргументов
    - Выберите нужный тест из меню

   Способ 2: Запуск через командную строку
    - huffman.exe test/test1.txt results/test1_encoded.bin results/test1_decoded.txt

3. ТЕСТОВЫЕ ФАЙЛЫ:
    - test1.txt - Простой английский текст
    - test2.txt - Текст с повторениями символов
    - test3.txt - Текст со спецсимволами
    - test4.txt - Пустой файл (тест на ошибку)
    - test5.txt - Большой файл (1000 строк)

4. РЕЗУЛЬТАТЫ:
    - Сжатые файлы сохраняются в папке results/*_encoded.bin
    - Декодированные файлы в папке results/*_decoded.txt
    - Статистика выводится в консоль

5. ПРОВЕРКА КОРРЕКТНОСТИ:
    - Программа автоматически сравнивает исходный и декодированный файлы
    - Результат сравнения выводится в консоль
    - Можно проверить вручную: fc test/test1.txt results/test1_decoded.txt
